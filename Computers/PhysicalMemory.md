# Memory
## Primary
- Directly accessed by CPU
### Random Access Memory
- Short term storage
- Volatile: Requires memory to maintain state

### Read Only Memory
- Permanent memory stored at time of manufacturing
- Non-volatile

## Secondary
- Accessed by loading into RAM
- Non-volatile

### Subtypes
- Hard Drive
- Solid State Drive
- Flash Drive
- Optical Drive
